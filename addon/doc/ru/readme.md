# Библия

Приложение предназначено для удобного изучения Священного Писания и предоставляет расширенные инструменты навигации, поиска и управления библейскими переводами. Это позволяет легко ориентироваться в тексте, перемещаться между книгами, главами и стихами, работать с разными переводами и сравнивать библейские отрывки.

## Основные возможности

Приложение позволяет:

- Использовать систему вкладок для одновременной работы с несколькими текстами Писания.
- Быстро перемещаться между книгами, главами, стихами и переводами.
- Копировать стихи и фрагменты с ссылкой.
- Просматривать параллельные места и схожие по контексту отрывки.
- Использовать поиск по тексту с возможностью фильтрации по книгам, главам или стихам, включая интеллектуальный поиск с помощью Gemini API.
- Работать с планами чтения, отслеживать прогресс и отмечать прочитанные места.
- Загружать и удалять переводы для разных языков и планы чтения.

### Основное окно

Основное окно содержит:

- **Текстовое поле** с текстом Библии, занимающее большую часть экрана.
- **Три выпадающих списка** для выбора книги, главы и перевода.

Содержимое текстового поля соответствует выбранным параметрам. В заголовке окна отображается текущая книга, глава и перевод, что помогает быстро определить, какой именно отрывок открыт.

**Открытие окна:** `NVDA+X`

Примечание: При первом запуске приложения автоматически открывается окно настроек. Для начала работы необходимо загрузить хотя бы один перевод Библии, после чего закрыть настройки и запустить приложение повторно.

**Навигация и управление:**

- **Переключение между выпадающими списками и текстом:** `Tab` / `Shift+Tab`
- **Переход к следующей / предыдущей книге:** `B` / `Shift+B`
- **Переход к следующей / предыдущей главе:** `C` / `Shift+C`
- **Переход к следующему / предыдущему переводу:** `T` / `Shift+T`
- **Перемещение курсора на 5 стихов назад / вперёд:** `PageUp` / `PageDown`
- **Перемещение курсора на 10 стихов назад / вперёд:** `Ctrl+PageUp` / `Ctrl+PageDown`
- **Быстрый переход к стиху:** цифры `0–9`

**Управление вкладками:**

- **Создать новую вкладку:** `Ctrl+T`
- **Закрыть текущую вкладку:** `Ctrl+W`
- **Переключиться на следующую / предыдущую вкладку:** `Ctrl+Tab` / `Ctrl+Shift+Tab`
- **Перейти на вкладку 1–9:** `Ctrl+1` .. `Ctrl+9`

**Настройки отображения:**

- **Показать / скрыть номера стихов:** `Ctrl+H`
- **Увеличить / уменьшить размер шрифта:** `Ctrl++` / `Ctrl+-`

**Другие клавиши:**

- **Поиск:** `Ctrl+F`
- **Планы чтения:** `Ctrl+R`
- **Переход по ссылке:** `Ctrl+L`
- **Закрыть окно:** `Alt+F4` или `Esc`
- **Справка:** `F1`

### Окно поиска

Окно поиска предназначено для поиска библейских стихов по тексту во всех книгах или в выбранных главах. Поддерживается фильтрация по книгам, поиск по целому слову, с учётом регистра и с использованием регулярных выражений. Также доступен интеллектуальный поиск с помощью Gemini API.

**Основные элементы:**

- **Поле для ввода поискового запроса**
- **Флажки параметров поиска** (целое слово, чувствительность к регистру, регулярные выражения)
- **Список результатов поиска**

**Клавиши управления:**

- **Переключение между элементами окна:** `Tab` / `Shift+Tab`
- **Открыть выбранный результат в основном окне:** `Enter`
- **Открыть выбранный результат в новой вкладке:** `Ctrl+Enter`
- **Закрыть окно поиска:** `Esc`

---

### Окно чтения планов

Окно чтения планов предназначено для работы с планами чтения Библии и отслеживания прогресса. Оно позволяет отмечать прочитанные дни и отрывки, переключаться между днями плана, изменять переводы и просматривать содержимое каждого дня.

**Основные элементы:**

- **Выпадающий список дней** для выбора конкретного дня плана.
- **Список чтения** для выбранного дня.
- **Текстовое поле** для отображения текста выбранного отрывка.

**Клавиши управления:**

- **Переключение между элементами окна:** `Tab` / `Shift+Tab`
- **Отметить день или отрывок как прочитанный / непрочитанный:** `Пробел`
- **Изменить перевод:** `T` / `Shift+T`
- **Изменить план чтения:** `Ctrl+Tab` / `Ctrl+Shift+Tab`
- **Переместить курсор на 5 стихов назад / вперёд:** `PageUp` / `PageDown`
- **Переместить курсор на 10 стихов назад / вперёд:** `Ctrl+PageUp` / `Ctrl+PageDown`
- **Перейти к стиху:** цифры `0–9`
- **Увеличить размер шрифта:** `Ctrl++`
- **Уменьшить размер шрифта:** `Ctrl+-`
- **Показать / скрыть номера стихов:** `Ctrl+H`
- **Закрыть окно:** `Esc`
- **Справка:** `F1`

### Окно перехода по ссылке и создания новой вкладки

Эти окна предназначены для быстрого перехода к конкретной книге, главе или стиху путём ввода библейской ссылки, а также для открытия новой вкладки с заданными параметрами. Поддерживаются различные форматы ввода ссылок, включая сокращённые названия книг, номера глав и стихов. Перечень доступных сокращений приведён в справке приложения.

Принимаются как восточный формат записи (например, «Ин. 3:16»), так и западный формат («Ин 3, 16»).

**Основные элементы:**

- **Поле для ввода ссылки** (сохраняет историю последних переходов).
- **Кнопка по умолчанию** (открывает Бытие, 1 глава).
- **Кнопка подтверждения перехода**.

**Клавиши управления:**

- **Перейти по введённой ссылке:** `Enter`
- **Закрыть окно:** `Esc`
- **Открыть справку:** `F1`

---

### Окно параллельных ссылок

Окно параллельных ссылок предназначено для просмотра и анализа параллельных мест и схожих по контексту библейских отрывков для выбранного стиха. Это помогает глубже понять смысл текста и связи между разными частями Священного Писания.

**Основные элементы:**

- **Выпадающий список параллельных ссылок**.
- **Текстовое поле** для отображения выбранного параллельного отрывка.

**Клавиши управления:**

- **Переключение между списком параллельных ссылок и текстом:** `Tab` / `Shift+Tab`
- **Открыть выбранную параллельную ссылку в основном окне:** `Enter`
- **Открыть выбранную параллельную ссылку в новой вкладке:** `Ctrl+Enter`
- **Переместить курсор на 5 стихов назад / вперёд:** `PageUp` / `PageDown`
- **Переместить курсор на 10 стихов назад / вперёд:** `Ctrl+PageUp` / `Ctrl+PageDown`
- **Перейти к стиху:** цифры `0–9`
- **Показать / скрыть номера стихов:** `Ctrl+H`
- **Закрыть окно:** `Esc`

### Меню программы

Меню программы в основном дублирует функции горячих клавиш и предназначено для удобства пользователей, которые не помнят все комбинации клавиш. Оно доступно в основном окне и в окне чтения планов.

**Основные возможности меню:**

- Выполнение действий, доступных через горячие клавиши.
- Управление параметрами отображения и вкладками.
- Доступ к настройкам программы и справке.

---

### Контекстное меню

Контекстное меню позволяет быстро выполнять основные действия с выделенным текстом или текущим стихом в тексте Библии.

Контекстное меню вызывается из основного окна с помощью:

- `Shift+F10` или клавиши контекстного меню на клавиатуре.

**Основные функции контекстного меню:**

- **Копировать** выделенный текст или текущий стих вместе с библейской ссылкой.
- **Открыть список параллельных ссылок** для текущего стиха.

### Настройки

Раздел настроек предназначен для управления основными параметрами программы, включая переводы, планы чтения, интеграцию с Gemini API и параметры обновления.

**Основные возможности:**

- **Управление переводами** (загрузка и удаление).
- **Управление планами чтения** (загрузка, удаление и сброс прогресса).
- **Ввод ключа API** для включения интеллектуального поиска.
- **Включение и отключение** автоматической проверки обновлений.

**Как открыть настройки:**

- Через меню программы: **Инструменты → Настройки**.
- Через настройки NVDA: **NVDA+N → Параметры → Настройки → Библия**.
