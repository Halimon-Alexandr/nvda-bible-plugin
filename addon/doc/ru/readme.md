Добро пожаловать в наше приложение — вашего надёжного помощника в изучении Слова Божьего!

Библия — это инструмент, созданный для работы с текстами Священного Писания.

Однако, в отличие от большинства подобных программ, наше приложение предлагает полностью доступный интерфейс, оптимизированный для невизуального доступа.

Оно состоит из главного окна, предназначенного для отображения текста Писания, а также элементов управления для изменения книги, главы и перевода.

Помимо основного окна, приложение содержит несколько дополнительных диалоговых окон, которые предоставляют дополнительные возможности — в частности, быстрый поиск нужных слов или фраз в тексте, переход по ссылке, а также просмотр параллельных мест Писания.

Окно программы открывается с помощью комбинации клавиш **NVDA+x**.

При **первом запуске** приложения автоматически открывается окно настроек. В этом окне необходимо выбрать **хотя бы один перевод Библии** из списка доступных вариантов и нажать кнопку **«ОК»**, чтобы сохранить выбор и продолжить работу.

> **Важно:** 

Без выбора хотя б одного перевода работа с текстами Библии будет невозможна.

В дальнейшем это окно можно будет открыть через **настройки NVDA**:

Меню NVDA (**NVDA+N** → **Параметры** → **Настройки** → **Библия**).

### Главное окно

Оно содержит четыре основных элемента: большое текстовое поле для отображения текста Библии, занимающее большую часть экрана, и три выпадающих списка для выбора книги, главы и перевода.

Кроме того, главное окно программы может содержать дополнительные вкладки, в каждой из которых можно независимо открыть другую книгу, главу или перевод.

В текстовом поле отображается выбранный отрывок Библии в соответствии с заданными параметрами, а в заголовке окна  название книги, номер главы, текущий перевод и номер вкладки, что помогает ориентироваться в текущем состоянии программы.

#### Навигация по тексту

Для перемещения по тексту используются стандартные клавиши навигации,

а также дополнительные комбинации для навигации по стихам:

- **PageDown** и **PageUp** — перемещают курсор на 5 стихов вперёд или назад.
- **Ctrl+PageDown** и **Ctrl+PageUp** — перемещают курсор на 10 стихов вперёд или назад.
- Нажатие любой цифры в текстовом поле перемещает курсор на указанный стих. Если необходимо переместить курсор на стих, состоящий из двух цифр (например, 12 или 45), нужно быстро нажать соответствующие клавиши на клавиатуре.

Для перехода между элементами интерфейса (текстовым полем и списками выбора) можно использовать:

- **Tab** — переход к следующему элементу.
- **Shift+Tab** — переход к предыдущему элементу.

Также доступны горячие клавиши для изменения книги, главы и перевода:

- **B** — следующая книга
- **Shift+B** — предыдущая книга
- **C** — следующая глава
- **Shift+C** — предыдущая глава
- **T** — следующий перевод
- **Shift+T** — предыдущий перевод

#### Контекстное меню

Чтобы открыть контекстное меню для выбранного стиха, нажмите клавишу **Applications** или **Shift+F10**.

В контекстном меню доступны следующие действия:

- Просмотреть **параллельные места**.
- **Скопировать** текущий стих со ссылкой или диапазон стихов, если выделено несколько.

#### Другие комбинации клавиш

- **Ctrl+F** — открыть окно поиска в Библии.
- **Ctrl+L** — перейти к месту по ссылке.
- **Ctrl+T** — создать новую вкладку.
- **Ctrl+1–9** — перейти на вкладку с соответствующим номером.
- **Ctrl+Tab** / **Ctrl+Shift+Tab** — циклически переключаться между вкладками.
- **Ctrl+F4** или **Ctrl+W** — закрыть текущую вкладку.
- **Alt+F4** или **Esc** — закрыть главное окно программы.


### Окно поиска

Окно поиска содержит больше элементов управления, чем главное окно, что позволяет гибко настраивать поиск по тексту Библии.

#### Элементы управления:
- **Текстовое поле для результатов поиска** — отображает найденные результаты после ввода запроса.
- **Поле для ввода поискового запроса** — помимо ввода текста, сохраняет последние 10 запросов, что позволяет быстро к ним вернуться.
- **Выпадающий список категорий поиска** — доступные категории: «Все книги», «Ни одной книги», «Книги Ветхого Завета», «Книги Нового Завета». Выбор категории автоматически отмечает соответствующую группу книг в списке книг.
- **Выпадающий список книг Библии** — позволяет выбрать конкретные книги для поиска. По умолчанию количество выбранных книг зависит от выбранной категории.

#### Дополнительные элементы поиска:
- **Флажки для дополнительных опций поиска:**
- **Поиск по целому слову** — активирует поиск только по полным словам.
- **С учётом регистра** — включает чувствительность к регистру символов.
- **Регулярное выражение** — позволяет использовать регулярные выражения для более гибкого поиска.
- **Интеллектуальный поиск** — использует модель Gemini для улудшенного поиска по смыслу. Требуется API-ключ.

- **Кнопка «Поиск»** — запускает поиск по тексту. Поиск также можно начать, нажав клавишу Enter в поле для ввода запроса.

#### Дополнительные функции:

Нажатие клавиши Enter на выбранном результате в поле результатов поиска открывает этот текст в текущей вкладке главного окна программы, а Ctrl+Enter — в новой вкладке, автоматически закрывая окно поиска.

- **Окно поиска можно закрыть также с помощью комбинации клавиш **Alt+F4** или **Esc**.**

### Окно параллельных ссылок

Окно параллельных ссылок позволяет просматривать библейские отрывки, которые тематически или контекстно соответствуют стиху, на котором было вызвано это окно.

Оно открывается через контекстное меню соответствующего стиха.
И состоит из двух основных элементов:

**Список ссылок** — содержит перечень библейских мест, которые являются параллельными текущему стиху.

При изменении выбранного элемента в списке ниже автоматически обновляется текст главы, содержащей соответствующую ссылку.

**Текстовое поле** — отображает полный текст главы, в которой расположен выбранный стих.

Фокус автоматически устанавливается на соответствующий стих, чтобы его можно было сразу прочитать.

В текстовом окне доступна навигация по тем же клавишам, что и в главном окне программы:

- **PageDown / PageUp** — перемещение на 5 стихов вперёд или назад.
- **Ctrl+PageDown / Ctrl+PageUp** — перемещение на 10 стихов вперёд или назад.
- Ввод цифры или нескольких цифр — переход к стиху с соответствующим номером.

В заголовке окна отображается место Писания, к которому относятся текущие параллельные ссылки, а также количество найденных ссылок.

Для открытия выбранного места можно нажать:

- **Enter** — открывает выбранную ссылку в текущей вкладке.
- **Ctrl+Enter** — открывает выбранную ссылку в новой вкладке.

Окно можно закрыть с помощью клавиш **Esc** или **Alt+F4**.

### Окно перехода по ссылке

Окно перехода по ссылке открывается комбинациями клавиш **Ctrl+L** или **Ctrl+T**.

Разница между ними лишь в том, что первая команда открывает введённую ссылку в текущей вкладке, тогда как вторая — в новой вкладке.

Окно состоит из следующих элементов:
**Поле для ввода ссылки на стих** — принимает ссылку на стих в виде сокращений.

Поддерживает историю последних 10 переходов, что позволяет быстро возвращаться к недавно открытым местам в тексте.

Принимаются как восточный формат («Ин. 3:16»), так и западный («Ин 3, 16»).

- Кнопка **«По умолчанию»** открывает начальное место — Бытие, глава 1, стих 1.
- Кнопка **«Открыть»** открывает указанную ссылку.
- Кнопка **«Отмена»** закрывает окно без выполнения перехода.

Окно также можно закрыть с помощью клавиш **Alt+F4** или **Esc**.
