# Біблія

Додаток призначений для зручного дослідження Святого Письма та надає розширені інструменти навігації, пошуку й керування біблійними перекладами. Що дозволяє легко орієнтуватися в тексті, переходити між книгами, розділами й віршами, працювати з різними перекладами та порівнювати біблійні місця.

## Основні можливості

Додаток дозволяє:

- Використовувати систему вкладок для одночасної роботи з кількома текстами Писання.

- Швидко переміщатися між книгами, розділами, віршами та перекладами.

- Копіювати вірші та фрагменти з посиланням.

- Переглядати паралельні місця та схожі за контекстом фрагменти.

- Використовувати пошук за текстом з можливістю фільтрації за книгами, розділами або віршами, включаючи інтелектуальний пошук за допомогою Gemini API.

- Працювати з планами читання, відстежувати прогрес та позначати прочитані місця.

- Завантажувати та видаляти переклади для різних мов та плани читання.

### Основне вікно

Основне вікно містить:

- **Текстове поле** з текстом Біблії, яке займає більшу частину екрана.
- **Три випадаючі списки** для вибору книги, розділу та перекладу.

Вміст текстового поля відповідає вибраним параметрам. У заголовку вікна відображається поточна книга, розділ і переклад, що допомагає швидко визначити, який саме уривок відкрито.

**Відкриття вікна:** `NVDA+X`

Примітка: Під час першого запуску додатку автоматично відкривається вікно налаштувань. Для початку роботи необхідно завантажити хоча б один переклад Біблії, після чого закрити налаштування та запустити додаток повторно.

**Навігація та керування:**

- **Перемикання між випадаючими списками та текстом:** `Tab` / `Shift+Tab`
- **Перехід до наступної / попередньої книги:** `B` / `Shift+B`
- **Перехід до наступного / попереднього розділу:** `C` / `Shift+C`
- **Перехід до наступного / попереднього перекладу:** `T` / `Shift+T`
- **Переміщення курсора на 5 віршів назад / вперед:** `PageUp` / `PageDown`
- **Переміщення курсора на 10 віршів назад / вперед:** `Ctrl+PageUp` / `Ctrl+PageDown`
- **Швидкий перехід до вірша:** цифри `0–9`

**Керування вкладками:**

- **Створити нову вкладку:** `Ctrl+T`
- **Закрити поточну вкладку:** `Ctrl+W`
- **Перемкнутися на наступну / попередню вкладку:** `Ctrl+Tab` / `Ctrl+Shift+Tab`
- **Перейти до вкладки 1–9:** `Ctrl+1` .. `Ctrl+9`

**Налаштування перегляду:**

- **Показати / приховати номери віршів:** `Ctrl+H`
- **Збільшити / зменшити розмір шрифту:** `Ctrl++` / `Ctrl+-`

**Інші клавіші:**

- **Пошук:** `Ctrl+F`
- **Плани читання:** `Ctrl+R`
- **Перехід за посиланням:** `Ctrl+L`
- **Закрити вікно:** `Alt+F4`, або `Esc`
- **Довідка:** `F1`

### Вікно пошуку

Вікно пошуку призначене для пошуку біблійних віршів за текстом у всіх книгах або в обраних розділах. Підтримується фільтрація за книгами, пошук за цілим словом, з урахуванням регістру та з використанням регулярних виразів. Також доступний інтелектуальний пошук за допомогою Gemini API.

**Основні елементи:**

- **Поле для введення пошукового запиту**
- **Прапорці параметрів пошуку** (ціле слово, чутливість до регістру, регулярні вирази)
- **Список результатів пошуку**

**Клавіші керування**

- **Перемикання між елементами вікна:** `Tab` / `Shift+Tab`
- **Відкрити вибраний результат в основному вікні:** `Enter`
- **Відкрити вибраний результат у новій вкладці:** `Ctrl+Enter`
- **Закрити вікно пошуку:** `Esc`

---

### Вікно читання планів

Вікно читання планів призначене для роботи з планами читання Біблії та відстеження прогресу. Воно дозволяє позначати прочитані дні й уривки, перемикатися між днями плану, змінювати переклади та переглядати вміст кожного дня.

**Основні елементи:**

- **Випадаючий список днів** для вибору конкретного дня плану.
- **Список читання** для обраного дня.
- **Текстове поле** для відображення тексту вибраного уривка.

---
*Клавіші керування**

- **Перемикання між елементами вікна:** `Tab` / `Shift+Tab`
- **Позначити день або уривок як прочитаний / непрочитаний:** `Пробіл`
- **Змінити переклад:** `T` / `Shift+T`
- **Змінити план читання:** `Ctrl+Tab` / `Ctrl+Shift+Tab`
- **Перемістити курсор на 5 віршів назад / вперед:** `PageUp` / `PageDown`
- **Перемістити курсор на 10 віршів назад / вперед:** `Ctrl+PageUp` / `Ctrl+PageDown`
- **Перейти до вірша:** цифри `0–9`
- **Збільшити розмір шрифту:** `Ctrl++`
- **Зменшити розмір шрифту:** `Ctrl+-`
- **Показати / приховати номери віршів:** `Ctrl+H`
- **Закрити вікно:** `Esc`
- **Довідка:** `F1`

### Вікно переходу за посиланням та створення нової вкладки

Ці вікна призначені для швидкого переходу до конкретної книги, розділу або вірша шляхом введення біблійного посилання, а також для відкриття нової вкладки з заданими параметрами. Підтримуються різні формати введення посилань, зокрема скорочені назви книг, номери розділів і віршів. Перелік доступних скорочень наведено в довідці додатку.

Приймаються як східний формат запису (наприклад, «Ів. 3:16»), так і західний формат («Ів 3, 16»).

**Основні елементи:**

- **Поле для введення посилання** (зберігає історію останніх переходів).
- **Кнопка за замовчуванням** (відкриває Буття, 1 розділ).
- **Кнопка підтвердження переходу**.

**Клавіші керування**

- **Перейти за введеним посиланням:** `Enter`
- **Закрити вікно:** `Esc`
- **Відкрити довідку:** `F1`

### Вікно паралельних посилань

Вікно паралельних посилань призначене для перегляду та аналізу паралельних місць і схожих за контекстом біблійних уривків для вибраного вірша. Це допомагає глибше зрозуміти зміст тексту та зв’язки між різними частинами Святого Письма.

**Основні елементи:**

- **Випадаючий список паралельних посилань**.
- **Текстове поле** для відображення вибраного паралельного уривка.

**Клавіші керування**

- **Перемикання між списком паралельних посилань і текстом:** `Tab` / `Shift+Tab`
- **Відкрити вибране паралельне посилання в основному вікні:** `Enter`
- **Відкрити вибране паралельне посилання в новій вкладці:** `Ctrl+Enter`
- **Перемістити курсор на 5 віршів назад / вперед:** `PageUp` / `PageDown`
- **Перемістити курсор на 10 віршів назад / вперед:** `Ctrl+PageUp` / `Ctrl+PageDown`
- **Перейти до вірша:** цифри `0–9`
- **Показати / приховати номери віршів:** `Ctrl+H`
- **Закрити вікно:** `Esc`

### Меню програми

Меню програми переважно дублює функції гарячих клавіш і призначене для зручності користувачів, які не пам’ятають усі комбінації клавіш. Воно доступне в основному вікні та у вікні читання планів.

**Основні можливості меню:**

- Виконання дій, доступних через гарячі клавіші.
- Керування параметрами перегляду та вкладками.
- Доступ до налаштувань програми та довідки.

---

### Контекстне меню

Контекстне меню дозволяє швидко виконувати основні дії з виділеним текстом або поточним віршем у тексті Біблії.

Контекстне меню викликається з основного вікна за допомогою:

- `Shift+F10` або клавіші контекстного меню на клавіатурі.

**Основні функції контекстного меню:**

- **Копіювати** виділений текст або поточний вірш разом із біблійним посиланням.
- **Відкрити список паралельних посилань** для поточного вірша.

---

### Налаштування

Розділ налаштувань призначений для керування основними параметрами програми, зокрема перекладами, планами читання, інтеграцією з Gemini API та параметрами оновлення.

**Основні можливості:**

- **Керування перекладами** (завантаження та видалення).
- **Керування планами читання** (завантаження, видалення та скидання прогресу).
- **Введення ключа API** для увімкнення інтелектуального пошуку.
- **Увімкнення та вимкнення** автоматичної перевірки оновлень.

**Як відкрити налаштування:**

- Через меню програми: **Інструменти → Налаштування**.
- Через налаштування NVDA: **NVDA+N → Параметри → Налаштування → Біблія**.
